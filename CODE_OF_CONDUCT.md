# Code of Conduct

## Our Pledge

We want Deferlytics to be a practical, respectful project for people building
faster analytics integrations. Everyone participating in this project is
expected to communicate with care, assume good intent where possible, and keep
technical discussion focused on improving the software.

## Expected Behavior

- Use welcoming and inclusive language.
- Be respectful of different experience levels and viewpoints.
- Give and accept constructive feedback gracefully.
- Keep issues, pull requests, and discussions focused on the project.
- Disclose conflicts of interest when they may affect a recommendation.

## Unacceptable Behavior

- Harassment, threats, intimidation, or personal attacks.
- Demeaning comments about identity, background, ability, or experience.
- Publishing private information without explicit permission.
- Spam, sustained off-topic posting, or deliberate disruption.
- Any conduct that would reasonably make others feel unsafe or unwelcome.

## Enforcement

Project maintainers may remove comments, close issues or pull requests, or block
participants when behavior violates this code of conduct. Enforcement decisions
should be proportionate, documented when practical, and focused on keeping the
project healthy.

## Reporting

If you need to report a conduct issue, contact the maintainers privately using
the repository owner's GitHub profile or the security reporting channel if the
issue also involves sensitive information. Reports will be handled with as much
confidentiality as the situation allows.
